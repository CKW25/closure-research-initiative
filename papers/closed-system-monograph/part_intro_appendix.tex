\chapter*{Orientation to the Appendices}
\phantomsection
\label{part:intro-appendix}
% ============================================================
\begin{chapterorientation}
\orientationitem{Settled}{The main text has already fixed the theorem chain and the status of conditional numerical and realization-level claims.}
\orientationitem{Task}{The appendices separate external background results, minimal examples, completion bookkeeping, and a downstream transport application.}
\orientationitem{Handoff}{The output is reference material for standard inputs, failure modes, boundary conditions, and later applications of the transport package.}
\end{chapterorientation}

The appendices do not change the dependency order of the main text.
\Cref{app:standard-results} records the standard external theorem
families used by the main text.  \Cref{app:minimal-examples} gives
small examples and failure modes for the main definitions.
\Cref{app:conditional-completion} records a completion ledger for
entries that are structurally reduced but not fixed by closed transport
alone.  \Cref{app:fusion} gives a downstream application of the
transport package to nonperturbative adiabatic invariants, while keeping
the intrinsic scalar statement separate from later interpretive
readings.
